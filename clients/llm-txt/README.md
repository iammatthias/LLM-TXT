# LLM-TXT Homepage

Homepage and documentation for the LLM-TXT project.

**Live:** [llm-txt.fun](https://llm-txt.fun)

## Features

- Project overview and platform links
- API documentation with parameter reference
- SDK usage examples
- Pricing information

## Development

```bash
# Start development server
bun run dev

# Build for production
bun run build

# Deploy
bun run deploy
```
